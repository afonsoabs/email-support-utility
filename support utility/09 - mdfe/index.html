Chegamos em um ponto muito importante para o pessoal que realiza transporte interestadual. O MDF-e é um documento de uso obrigatório nessa transição de carga entre estados, pois caso chegue até uma parada de fiscalização e o motorista não possua o documento, a carga será apreendida.
MDF-e - Como realizar o seu processo de emissão

Primeiramente, você deve ler o email que enviamos antes desse a respeito de como realizar o romaneio, pois para a emissão do manifesto, o gatinho utilizado é o fechamento do romaneio.

Ao fechar o romaneio, é possível acessá-lo na tela de gerenciamento de manifestos conforme mostra a imagem abaixo:



Mas, antes de emitir, é necessário que se realizem alguns cadastros. Vamos lá.
Cadastros obrigatórios para emissão de MDF-e
Basicamente são dois cadastros obrigatórios, sendo eles:

  • Transportadora(s) (cadastros> expedição > transportadoras);


Preenchendo os campos em negrito que são obrigatórios para o sistema e em amarelo para que a transportadora se enquadre nas Tags do XML do MDF-e.

  • Morotista(s) (cadastros > expedição > motoristas)


Sendo obrigatório apenas os campos em negrito (Nome e CPF).
