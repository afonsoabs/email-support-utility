Vamos aprender a configurar o backup?
O backup é um processo muito simples pois necessita que você clique em apenas 1 botão!

  • 1º - No servidor, acesse Utilidades > Backup Sistema.

  • 2º - Se estiver marcada, Desmarque a opção Somente específico.

  • 3º - Em "Pasta para backup 1" e/ou "Pasta para backup 2", clique na pasta ao lado para selecionar onde salvar o seu backup.

  • 4º - Clique no botão Realizar bkp e aguarde a finalização do processo.
Abaixo tem uma imagem mostrando exatamente como chegar na tela para realizar o seu backup:

Gostaria de lembrar apenas três coisas:

  • 1º - Sempre faça o backup em um HD externo ou pendrive e guarde em local seguro.

  • 2º - Faça diariamente esse processo, para que seus dados estejam sempre atualizados.

  • 3º - NUNCA salve o backup em uma pasta no servidor, pois em caso de perda, o backup vai junto :(
