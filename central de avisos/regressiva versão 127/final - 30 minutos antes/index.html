Falta pouco para uma grande atualização no sistema ADS!
Daqui a alguns minutos, você poderá realizar a atualização do sistema e recepcionar a versão 127, onde agora vamos te lembrar sempre que atualizações importantes forem lançacadas. Mas antes disso, vamos relembrar algumas coisas que devem ser feitas antes:

  • Todos os computadores devem ser atualizados, incluindo o servidor;

  • Faça uma cópia do seu certificado digital;

  • Faça um backup do sistema ( utilidades > backup sistema );


Importante: Após atualizar para a versão 127, verifique se a impressora de etiqueta está funcionando corretamente, assim como o seu certificado e a folha de separação.
