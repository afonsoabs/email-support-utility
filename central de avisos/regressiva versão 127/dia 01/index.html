Isso mesmo, o sistema ADS está mudando para melhor!
No dia 08/05/2023 (segunda-feira) será lançada a versão 127 do sistema, com um compilado de melhorias, correções de bugs e novidades para vocês!
Mas, antes de atualizar precisamos ressaltar alguns pontos:

  • Todos os computadores devem ser atualizados, incluindo o servidor;

  • Faça uma cópia do seu certificado digital;

  • Faça um backup do sistema ( utilidades > backup sistema );

  • Para atualizar, basta ir em Ajuda > atualização do sistema
O mesmo começará a atualização e quando finalizar, vai abrir a tela de login/senha do sistema ads novamente. Feito isso o processo da primeira etapa está concluído!

Importante: Lembre-se de fazer isso em todas as máquinas. Caso dê erro na hora de atualizar, contate o suporte.
