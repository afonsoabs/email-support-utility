Sistema ADS Versão 133!
O que há de novo?
Estamos iniciando um novo mês e com ele, trazemos uma nova versão do sistema ads com diversas melhorias e correções! Hoje vamos explorar um pouco a respeito do que foi implementado e corrigido nesta nova versão.

Descrição dos produtos

Conforme diversas solicitações, aumentamos o campo de descrição (nome do produto) para 60 caracteres. Agora, é possível que você digite nomes maiores para os seus produtos sem a necessidade de abreviar.

DIFAL - cálculo automatizado

Antes, quando era necessário realizar uma venda para um cliente não contribuinte de fora do estado era necessário entrar no informe de impostos (opção T do faturamento) para que o sistema calculasse o diferencial de alíquota.
Agora, o sistema identifica esse tipo de operação e realiza o calculo automaticamente, não sendo mais necessário entrar no informe de impostos.

SICREDI - gerar mais de uma remessa/dia

Houve uma atualização nas políticas de remessa do banco e nós não ficamos para trás. Agora, é possível gerar mais de uma remessa por dia e enviar todas ao banco, coisa que antes era bloqueado a 1 remessa ao dia pelo próprio banco.

Compras - exportar e visualizar xml original

Agora no módulo de compras, atualizamos o sistema para que ele guarde o XML original das suas notas de compra para que posteriormente, seja possível visualizar o danfe do mesmo e exportá-lo do sistema.



Outra coisa que veio junto com a atualização desse módulo foi o envio dos XML de entrada para a contabilidade! Em decorrência do sistema guardar o xml original, agora é possível enviá-lo para o seu contador(a) assim como é realizado o processo de envio do faturamento.



Validação CST para débito PIS/COFINS

Diretamente no cadastro do produto em operações fiscais, o campo "debita pis/cofins" foi ajustado e está funcionando corretamente.
Vale ressaltar que o seu uso se dá da seguinte forma: para PIS/COFINS código 01 o campo deve ser marcado. Para qualquer outro tipo de código, deve ser desmarcado.



Pode ser ajustado pelo botão de ajuste geral também.
